Isometric Drawing Toolkit

The isometric drawing toolkit provides an API where different components can share the same 3-dimensional structure between them. This ranges from display components for viewing, to control component for manipulating structures.

An isometric drawing interface has been created as a tool for creating drawings, and as a wrapper application that demonstrates the integration of the various built-in components served by the API.

  • Controls such as the rotation buttons above;
Custom components can be added just like the built-in ones, interact with the store and all components will be updated automatically.
